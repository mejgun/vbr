# VBoxRec
  Unofficial VBox app

## Features

  - Watch and search epg and set for the record.
  - Set record even without epg, just like good old vhs recorder: time, channel, rec!
  - Works with multiple devices.
  
### Tech

Requied minimal: Android 4.1, VBox device.
