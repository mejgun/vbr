# vbr
  Unofficial vbox app

## Features

  - Watch and search epg and set for the record.
  - Set record even without epg, just like old good vhs recorder: time, channel, rec!
  - Works with multiply devices.
  
### Tech

Requied minimal: Android 4.4, VBox device.
